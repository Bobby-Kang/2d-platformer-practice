# 2D Platformer Practice

A 2D platformer game project built with Unity.

## Requirements

- Unity 6000.4.0f1
- Universal Render Pipeline (URP)

## Getting Started

1. Clone the repository
   ```bash
   git clone https://github.com/Bobby-Kang/2d-platformer-practice.git
   ```
2. Open the project in Unity Hub
3. Open `Assets/Scenes/SampleScene.unity` or `Assets/Simple 2D Platformer BE2/Demo/Demo.unity`

## Project Structure

```
Assets/
├── Scenes/              # Main scenes
├── Settings/            # URP render pipeline settings
├── Simple 2D Platformer BE2/  # Platformer assets & demo
│   ├── Demo/            # Demo scene
│   ├── Sprites/         # Character, enemy, and environment sprites
│   └── ReadMe/          # Documentation scripts
└── Sprites/             # Additional sprites
```
